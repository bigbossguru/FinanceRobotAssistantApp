# CryptoAlertApp
Crypto alert app, which send signals to telegram bot

## Manual
### Install Python version 3.7.9
### Install and activate virtual environment
```
python -m venv venv
```
```
venv\Scripts\activate.bat
```

### Install packages
```
pip install -r requirements.txt
```

### Run app
run app background
```
pythonw app.py
```
